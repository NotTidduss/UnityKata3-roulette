Unity Kata #3 - Roulette

Flow:
1. Bet x amount of chips on either red, black or 0
2. Wheel spins
3. Wheel stops - Results

Requirements:
1. UI for betting
2. Model / mish-mesh for wheel where numbers are collidable cubes
3. Physics for Wheel rotation and ball movement
4. UI for the results / play again
5. Score system

Goal:
Go All-In on Red and Win.
